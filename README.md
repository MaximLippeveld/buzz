# Development

## Setup environment
1. Install Miniconda3
1. Install mamba in base environment
1. Run mamba env create -f environment.yml
1. Install nvm (node version manager)
1. `nvm install 15`
1. `npm install`
1. `export FLASK_APP=$(pwd)/api/main.py`

## Setup servers
`npm run dev:serve`
In data directory `flask run`
